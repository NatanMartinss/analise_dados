📘 Capítulo 1 — Fundamentos da Análise de Dados

📊 O que são dados?

Dados são informações registradas sobre o mundo — números, textos, imagens, sons, tudo o que pode ser observado e armazenado. Quando você marca “curtir” em um vídeo, faz uma compra online ou mede a temperatura, está gerando dados.

💡 Dica: Dado é o “tijolo”. Informação é a “parede”. Sozinhos, os dados não têm sentido — mas organizados, formam conhecimento.

📜 Breve história da análise de dados

Desde as antigas civilizações, coletar dados era essencial. Egípcios registravam as cheias do Nilo, mercadores babilônios anotavam preços, e os romanos faziam censos populacionais.

No século XIX, a estatística surgiu como ciência. Já no século XX, com o avanço dos computadores, a análise de dados se tornou digital. Na década de 2000, com a internet, redes sociais e sensores, nasceu o conceito de Big Data.

🎯 Tipos de dados

  • Quantitativos: Números que podem ser medidos — idade, peso, faturamento.
  • Qualitativos: Categorias ou rótulos — gênero, estado civil, cor dos olhos.
  • Estruturados: Organizados em tabelas (planilhas, bancos de dados SQL).
  • Não estruturados: Textos, vídeos, imagens ou áudios — exigem IA para análise.
🎮 Atividade prática: Liste três tipos de dados que você gera em um único dia (ex: músicas ouvidas, passos dados, mensagens enviadas).

⚙️ O ciclo da análise de dados

  1. Coleta: capturar informações de diferentes fontes.
  2. Limpeza: remover erros e inconsistências.
  3. Análise: descobrir padrões e relações.
  4. Visualização: transformar resultados em gráficos e dashboards.
  5. Decisão: aplicar os insights em ações reais.
📈 Exemplo: Uma cafeteria pode usar dados de vendas e horários de pico para ajustar o número de funcionários — simples, mas poderoso!

🚀 Por que a análise de dados importa?

Porque o mundo gira em torno de informações. Empresas tomam decisões com base em dados, governos planejam políticas públicas, e até você usa dados para organizar suas finanças.

Saber interpretar dados é uma habilidade essencial — abre portas em qualquer área e ajuda a enxergar padrões que outros não veem.

🧭 Resumo do Capítulo

  • Dados são registros do mundo real.
  • Podem ser quantitativos, qualitativos, estruturados ou não estruturados.
  • A análise transforma dados em conhecimento útil.
  • O ciclo envolve coleta, limpeza, análise, visualização e decisão.
  • Usar dados é o caminho para decisões mais inteligentes.
