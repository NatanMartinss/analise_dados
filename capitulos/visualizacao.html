📊 Capítulo 4 — Visualização de Dados

🎨 A Arte de Contar Histórias com Dados

Visualizar dados é transformar números em narrativas. Um gráfico bem feito pode revelar padrões que passariam despercebidos em uma tabela. É aqui que a análise encontra o design — e você começa a ver os dados “falarem”.

💡 Regra de Ouro: seu gráfico deve contar uma história em até 3 segundos. Se a pessoa precisar pensar muito, algo precisa ser simplificado.

🎯 Escolhendo o Gráfico Certo

Nem todo dado deve virar gráfico de pizza, e nem toda linha conta uma boa história. Veja um guia rápido:

🎯 Objetivo 📈 Gráfico Ideal ⚠️ Evite se...
Comparar valores 📊 Barras Mais de 10 categorias
Mostrar evolução 📈 Linha Dados não temporais
Mostrar proporções 🥧 Pizza Mais de 5 fatias
Mostrar correlação 🔵 Dispersão Sem relação clara
Mostrar densidade 🌡️ Heatmap Poucos dados

📊 Gráficos de Barras

O tipo mais direto — perfeito para comparações entre categorias.


import matplotlib.pyplot as plt

categorias = ["Produto A", "Produto B", "Produto C"]
vendas = [3500, 2700, 4200]

plt.bar(categorias, vendas, color="skyblue")
plt.title("Vendas por Produto")
plt.ylabel("Unidades Vendidas")
plt.show()
🧠 Exercício: Crie um gráfico de barras mostrando o número de atendimentos por especialidade médica em um mês.

📈 Gráficos de Linha

Gráficos de linha são ideais para representar evolução ao longo do tempo.


meses = ["Jan", "Fev", "Mar", "Abr", "Mai", "Jun"]
vendas = [1000, 1200, 900, 1400, 1600, 2000]

plt.plot(meses, vendas, color="cyan", marker="o", linewidth=2)
plt.title("Evolução das Vendas")
plt.xlabel("Mês")
plt.ylabel("Vendas (R$)")
plt.grid(True)
plt.show()
💡 Dica: use plt.fill_between() para preencher a área entre a linha e o eixo — ótimo para destacar crescimento ou queda.

🥧 Gráficos de Pizza

Úteis para representar proporções de um todo, mas cuidado com o exagero.


categorias = ["Marketing", "RH", "Vendas", "TI"]
valores = [25, 15, 40, 20]

plt.pie(valores, labels=categorias, autopct="%1.1f%%", startangle=90)
plt.title("Distribuição Orçamentária")
plt.show()
🎯 Desafio: Mostre a porcentagem de dispositivos usados por usuários (Desktop, Mobile, Tablet). Dê destaque ao Mobile com o parâmetro explode.

🔵 Gráficos de Dispersão

Usado para mostrar relação entre duas variáveis numéricas — ótimo para encontrar padrões escondidos.


import numpy as np

x = np.random.randint(50, 150, 50)
y = x * np.random.uniform(0.8, 1.2, 50)

plt.scatter(x, y, color="violet", alpha=0.6)
plt.title("Correlação entre Variáveis")
plt.xlabel("Variável X")
plt.ylabel("Variável Y")
plt.show()
💡 Dica: use np.corrcoef(x, y) para medir o quanto as variáveis estão relacionadas.

🌡️ Heatmaps (Mapas de Calor)

Mapas de calor mostram a intensidade de valores em uma matriz — excelentes para visualização de correlação.


import seaborn as sns
import pandas as pd

dados = pd.DataFrame({
    "Vendas": np.random.randint(50, 150, 10),
    "Lucro": np.random.randint(10, 100, 10),
    "Satisfação": np.random.randint(1, 10, 10)
})

sns.heatmap(dados.corr(), annot=True, cmap="coolwarm")
plt.title("Correlação entre Variáveis")
plt.show()

🎨 Customizando Estilo

Um bom gráfico não é só funcional, mas também agradável visualmente. O Matplotlib permite temas e ajustes finos de cores e grades: 


plt.style.use("seaborn-v0_8-darkgrid")
plt.rcParams["figure.figsize"] = (10, 6)
plt.rcParams["axes.titlesize"] = 16
plt.rcParams["axes.labelcolor"] = "white"
plt.rcParams["text.color"] = "white"

Esses ajustes dão personalidade e coerência visual à sua análise.
