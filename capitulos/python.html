✨ Por que Python é o coração da análise de dados

Python é uma linguagem simples, poderosa e extremamente popular entre analistas e cientistas de dados. Ela nasceu em 1991, criada por Guido van Rossum, e hoje é a principal linguagem em áreas como automação, inteligência artificial e ciência de dados.

💡 Dica: Se você nunca programou, Python é o melhor ponto de partida. A sintaxe é quase "humana" — você vai ler e entender o que está acontecendo sem precisar decorar comandos obscuros.

🧩 Variáveis e tipos de dados

Variáveis são como “caixas” onde você guarda informações. Elas podem armazenar textos, números, listas, e até objetos inteiros. 


# Exemplo simples
nome = "Natan"
idade = 28
altura = 1.78
ativo = True

print(nome, idade, altura, ativo)

Tipos primitivos principais:

  • int → números inteiros
  • float → números decimais
  • str → textos (strings)
  • bool → valores lógicos (True/False)

🔁 Estruturas de controle

Programar é tomar decisões. O Python faz isso com estruturas condicionais (if) e repetições (for e while). 


idade = 20

if idade >= 18:
    print("Maior de idade")
else:
    print("Menor de idade")

Agora, veja um exemplo com for:


for numero in range(1, 6):
    print(f"Número {numero}")
💡 Dica: O range() não inclui o último número. range(1,6) vai de 1 a 5.

🧱 Estruturas de dados

Listas

Listas armazenam vários valores em uma única variável:


frutas = ["maçã", "banana", "laranja"]
print(frutas[0])  # maçã
frutas.append("uva")

Dicionários

Dicionários armazenam dados em pares chave: valor:


pessoa = { "nome": "Natan", "idade": 28 }
print(pessoa["nome"])
🎯 Desafio:
Crie uma lista de dicionários com 3 pessoas e suas idades, e mostre quem é o mais velho.

📂 Lendo arquivos

Um dos grandes poderes do Python é ler e escrever dados de diversos formatos, como CSV e JSON. 


import csv

with open("vendas.csv") as arquivo:
    leitor = csv.reader(arquivo)
    for linha in leitor:
        print(linha)

Com JSON é ainda mais simples:


import json

with open("dados.json") as arquivo:
    dados = json.load(arquivo)
    print(dados["clientes"])

📊 Análise com Pandas

O Pandas é a biblioteca mais usada para análise de dados em Python. Ele transforma tabelas em estruturas chamadas DataFrames, que você pode filtrar, ordenar e agrupar com facilidade. 


import pandas as pd

df = pd.read_csv("vendas.csv")
print(df.head())
print(df.describe())

Você pode filtrar dados assim: 


filtrado = df[df["Valor"] > 1000]
print(filtrado)

📈 Visualização com Matplotlib

Gráficos são essenciais para contar histórias com dados. Veja como criar um gráfico simples:


import matplotlib.pyplot as plt

vendas = [100, 200, 300, 250, 400]
meses = ["Jan", "Fev", "Mar", "Abr", "Mai"]

plt.plot(meses, vendas, color="cyan", marker="o")
plt.title("Evolução de Vendas")
plt.xlabel("Mês")
plt.ylabel("Vendas")
plt.show()
💡 Dica: Combine Pandas com Matplotlib e o poder da análise visual explode!

🚀 Mini Projeto Final: Analisando Vendas

Vamos aplicar tudo que aprendemos até aqui com um pequeno projeto de análise real.


import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("vendas.csv")
df["Lucro"] = df["Valor"] - df["Custo"]

lucro_mensal = df.groupby("Mês")["Lucro"].sum()

lucro_mensal.plot(kind="bar", color="orchid")
plt.title("Lucro Mensal")
plt.show()
🎯 Desafio Extra:
Adicione uma coluna com a margem de lucro (%), e identifique qual mês foi mais rentável.
