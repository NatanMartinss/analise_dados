🎯 Por que Estatística Importa

Imagine que você é gerente de vendas e precisa decidir:

  • Qual produto lançar primeiro?
  • Em que região investir mais?
  • Qual campanha de marketing funcionou melhor?
Sem estatística, você está apenas “chutando”. Com estatística, você decide com base em evidências. 🎯

💡 Mindset Estatístico: Não existe “sorte” ou “azar” nos negócios — existe padrão, tendência e probabilidade. Nosso papel é descobrir esses padrões.

📈 Medidas de Tendência Central

As medidas de tendência central resumem um conjunto de dados por um único valor representativo.

👨‍👩‍👧‍👦 A Família das Médias

As três mais conhecidas são:

  • Média: soma de todos os valores dividida pela quantidade de elementos.
  • Mediana: valor central, ideal quando há valores extremos (outliers).
  • Moda: o valor que mais se repete.

import numpy as np
from scipy import stats

salarios = [2000, 2500, 3000, 3200, 3500, 3800, 4000, 4200, 4500, 25000]

media = np.mean(salarios)
mediana = np.median(salarios)
moda = stats.mode(salarios, keepdims=True).mode[0]

print(f"📊 Média: R$ {media:,.2f}")
print(f"🎯 Mediana: R$ {mediana:,.2f}")
print(f"🔄 Moda: R$ {moda:,.2f}")

📏 Medidas de Dispersão

Enquanto as médias mostram o centro dos dados, as medidas de dispersão mostram o quanto eles se espalham.

📊 Desvio Padrão e Variância


empresa_a = [4800, 4900, 5000, 5100, 5200]  # Estável
empresa_b = [3000, 4000, 5000, 6000, 7000]  # Volátil

desvio_a = np.std(empresa_a)
desvio_b = np.std(empresa_b)

print(f"Empresa A: {desvio_a:.2f}")
print(f"Empresa B: {desvio_b:.2f}")
💡 Desvio padrão mostra o quanto os dados se afastam da média. Um desvio pequeno indica estabilidade; um grande indica alta variação.

📉 Coeficiente de Variação

Para comparar conjuntos diferentes, usamos o Coeficiente de Variação (CV): CV = (Desvio / Média) × 100.

📦 Quartis e Box Plot

Os quartis dividem o conjunto de dados em quatro partes. O gráfico Box Plot mostra visualmente essa divisão e destaca outliers.


import seaborn as sns
import pandas as pd

dados = {
  "TI": np.random.normal(8000, 2000, 50),
  "Vendas": np.random.normal(6000, 3000, 50),
  "RH": np.random.normal(5000, 1000, 50),
}

df = pd.DataFrame(dados)
sns.boxplot(data=df)
plt.title("Distribuição Salarial por Departamento")
plt.show()
💡 Interpretação rápida: - A linha no meio da caixa é a mediana. - As bordas da caixa representam o 1º e 3º quartil. - Pontos isolados fora dos “bigodes” são outliers.

🎲 Distribuições de Probabilidade

📈 Distribuição Normal — a “Rainha” das distribuições

A distribuição normal (ou curva de Gauss) é simétrica e representa a maioria dos fenômenos naturais. 


alturas = np.random.normal(170, 10, 1000)
plt.hist(alturas, bins=30, color="skyblue", alpha=0.7)
plt.title("Distribuição Normal de Alturas")
plt.xlabel("Altura (cm)")
plt.ylabel("Frequência")
plt.show()

Em uma distribuição normal:

  • 68% dos dados estão a ±1 desvio padrão da média
  • 95% a ±2 desvios
  • 99,7% a ±3 desvios

🏭 Aplicação Prática: Controle de Qualidade

Em fábricas, estatística é usada para detectar falhas de produção. 


peso_ideal = 500
tolerancia = 10
pesos = np.random.normal(peso_ideal, 5, 200)

validos = np.sum((pesos >= 490) & (pesos <= 510))
print(f"Percentual dentro da especificação: {validos/len(pesos)*100:.1f}%")
💡 Insight: Se a maioria das amostras está fora da faixa, o processo precisa ser recalibrado — estatística salva milhões em desperdício industrial!
